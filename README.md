# FOA - Portfolio

A modern frontend developer portfolio built to showcase my skills, services, and projects. The portfolio highlights who I am, the technologies I work with, and some of the web applications I have built.

## Live Demo

[View Portfolio](https://flourishagbanoma.vercel.app/)


## Built With

* React JS
* Tailwind CSS

## Features

* Responsive modern design
* About Me section
* Services section
* Projects showcase
* Smooth animations using React Bits
* Custom animated cursor
* Interactive UI and hover effects
